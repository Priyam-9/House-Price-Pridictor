#  House Price Predictor  
A Machine Learning project that predicts house prices using **Linear Regression, SVR, XGB, Random Forest** and provides for real-time predictions.

This project showcases the full ML lifecycle — data preprocessing, EDA, model building, saving the model.

---

##  Features
- Cleaned & preprocessed dataset  
- Exploratory Data Analysis (EDA)  
- Linear Regression model training  
- Model saved with Pickle  
- Modular and beginner-friendly code structure.  

---

##  Tech Stack
- **Python**  
- **Pandas**, **NumPy**  
- **Scikit-Learn**  
- **Matplotlib**, **Seaborn**  
- **Pickle**
